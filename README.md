# mcsr-paceman-analyzer
这是一个学生作业，是在paceman上获取数据做进一步数据分析的小玩意。我是一名初学者，很需要创作建议和批评指正！

This is a student assignment—a small tool for fetching data from paceman to perform further data analysis. As a beginner, I'm very much in need of creative suggestions and constructive criticism!

安装node，配置好node环境后，在命令行窗口定位到该文件夹路径，运行'npm install'

After installing Node.js and configuring the Node environment, navigate to the folder path in the command line window and run 'npm install'.

完成后再运行'node paceman_server.js'

After completion, run 'node paceman_server.js'

打开index.html，输入mc或者twitch名称即可查询（跟你在paceman.gg一样）

Open index.html, then enter either an MC or Twitch name to search (just like you would on paceman.gg).
